## 开发

```
# 开发
npm run dev

# 构建
npm run build

# 发布前预览
npm run prepublishOnly && arco preview

# 发布至物料平台（需先发布 NPM 包）
arco sync
```

## 安装

pnpm:

```bash
$ npm install -g pnpm
$ pnpm install --save @arco-vue-pro-components/pro-components
```
