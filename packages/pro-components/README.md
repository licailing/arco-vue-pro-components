## 开发

```
# 开发
npm run dev

# 构建
npm run build

# 发布前预览
npm run prepublishOnly && arco preview

# 发布至物料平台（需先发布 NPM 包）
arco sync
```

## 安装

npm:

```bash
$ npm install --save-dev @acro-vue-pro-components/pro-components
```

yarn:

```bash
$ yarn add @acro-vue-pro-components/pro-components
```
